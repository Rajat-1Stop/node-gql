# Express GraphQL App with Onion Architecture

This repository contains an Express application built with GraphQL using Apollo Client and follows the onion architecture.

## Prerequisites

- Node.js version: 18.18.0
- GraphQL version: 16.8.1
- "@apollo/client" version: ^3.8.8

## Installation

1. Clone the repository:

   ```bash
   git clone <repository-url.git>
   cd <repository-folder>
   ```
